# API 測試頁面使用說明

## 啟動方式

```bash
cd /workspace/project
python main.py
```

自動開啟瀏覽器後，手動訪問：
```
http://localhost:9999/test
```

## 頁面功能

### 1. 快速測試按鈕

| 按鈕 | 功能 | 預期回應時間 |
|------|------|-------------|
| **健康檢查** | 測試伺服器是否正常運作 | < 50ms |
| **分析台積電** | 測試 AI 綜合分析 | 3-10 秒 |
| **AI 聊天** | 測試聊天功能 | 1-3 秒 |
| **估值指標** | 測試 PE、PB 等估值 | 1-3 秒 |
| **產業比較** | 測試同業比較 | 5-15 秒 |
| **股利資料** | 測試除權息資料 | 1-3 秒 |

### 2. 自訂股票代碼

在「股票代碼」輸入框輸入股票代碼（例如：2330.TW、2317.TW、2454.TW），然後點擊想要測試的功能按鈕。

### 3. 測試特定端點

往下捲動到「所有 API 端點」區域，每個端點都有：
- 端點名稱（例如：AI 綜合分析）
- API 路徑（例如：GET /api/v1/analysis/ai/{stock_id}）
- [測試] 按鈕

## 測試結果解讀

```
狀態：成功 (200)    回應時間：156ms
```

- ✅ **狀態：成功 (200)** → API 正常運作
- ❌ **狀態：錯誤 (500)** → API 有問題
- ⏱ **回應時間** → 效能指標（< 1000ms 算快）

## 常見問題

### Q: 點擊按鈕後沒反應？

可能原因：
1. 伺服器沒啟動 → 執行 `python main.py`
2. 端口錯誤 → 確認是 9999
3. 網路問題 → 重新整理頁面

### Q: 回應時間很長？

正常情況：
- 健康檢查：< 50ms
- 估值指標：1-3 秒
- 綜合分析：3-10 秒
- ML 預測：5-15 秒

如果超過 30 秒，可能是：
1. Yahoo Finance 回應慢
2. ML 模型計算中
3. 網路連線問題

### Q: 看到錯誤訊息？

常見錯誤：
- **500 Internal Server Error** → 程式碼有問題，看伺服器日誌
- **404 Not Found** → 端點不存在
- **422 Validation Error** → 參數錯誤
